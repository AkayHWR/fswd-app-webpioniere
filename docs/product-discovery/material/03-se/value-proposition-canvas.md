# Value Proposition Canvas

## Customer Profile

### Customer Jobs

- Eine Studienfrage stellen, wenn offizielle Quellen nicht ausreichen.
- Erfahrungen zu Klausuren, Dozenten, Vertiefungen oder Bachelorarbeit finden.
- Einschaetzen, welche Antwort in der eigenen Situation hilfreich ist.
- Eigenes Wissen mit anderen Studierenden teilen.

### Pains

- Informationen sind in WhatsApp, Discord oder persoenlichen Kontakten verstreut.
- Gute Antworten sind spaeter kaum wiederzufinden.
- Fragen werden mehrfach gestellt.
- Antworten sind subjektiv und schwer zu bewerten.
- Unbekannte offene Foren wirken weniger vertrauenswuerdig.

### Gains

- Ein zentraler Ort fuer erfahrungsbasierte Studienfragen.
- Schneller Zugriff auf vorhandene Fragen und Antworten.
- Sichtbarkeit fuer hilfreiche Beitraege.
- Mehr Vertrauen durch HWR-Kontext.
- Weniger Wiederholung in privaten Chats.

## Value Map

### Products and Services

- StudySwap Web App
- HWR-Mail-Registrierung
- Dashboard
- Frage erstellen
- Antworten schreiben
- Hashtag-Filter
- Votes
- Loesung markieren
- Leaderboard

### Pain Relievers

| Pain | App-Element |
| --- | --- |
| Verstreute Informationen | Zentrale Frageuebersicht |
| Schlechte Wiederauffindbarkeit | Hashtags und Detailseiten |
| Unklare Antwortqualitaet | Votes und Loesungsmarkierung |
| Geringes Vertrauen | Zugriff ueber HWR-Mail |
| Wiederholte Fragen | Bestehende Fragen bleiben sichtbar |

### Gain Creators

| Gain | App-Element |
| --- | --- |
| Schneller Austausch | Einfache Frage- und Antwortfunktion |
| Community-Wissen | Alle Studierenden koennen lesen und beitragen |
| Anerkennung | Leaderboard und Votes |
| Orientierung | Geloeste Fragen und hilfreiche Antworten |

## Fit Statement

StudySwap passt zur Zielgruppe, weil die App kein allgemeines Wissensproblem
loest, sondern einen lokalen Austausch organisiert: Studierende beantworten
Fragen, die nur andere Studierende aus demselben Kontext wirklich einschaetzen
koennen.
