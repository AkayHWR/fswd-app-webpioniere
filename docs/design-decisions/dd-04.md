# 04: Dynamic Categorization vs. Fixed Category List

## Meta
Status : **Decided**
Updated : 16-Jun-2026

## Problem Statement
The platform needs a way to categorize questions so that users can effectively filter and find relevant topics on the dashboard. We had to decide whether to restrict users to a predefined list of categories (e.g., official study modules, semesters) or allow them to freely define their own categories using hashtags.

## Decision
We decided to go with Option 1 (Dynamic Categorization via Free Hashtags).
*Decision was taken by:* github.com/KarlTebbe

## Regarded options

### Option 1: Dynamic categorization via free hashtag entry
With this option, users can create and assign any hashtags they want when formulating their question, without being restricted to a set list.

Pros:
- Maximum flexibility: The platform adapts organically to new topics, specific niches, or new modules without requiring admin updates.
- Highly specific: Users can tag exactly what their problem is about (e.g., `#BürgerlichesRecht` or a specific professor's name).
- Lower maintenance: No need to constantly maintain and update a static database table of allowed categories.

Cons:
- Prone to inconsistencies: Users might use different variations of the same tag (e.g., `#finance` vs. `#Finanzen` vs. `#Finance1`), which requires backend normalization (like lowercase conversion and duplicate filtering) to ensure the dashboard filter works properly.

### Option 2: Predefined list of categories (Dropdown)
With this option, users must select one or multiple categories from a fixed dropdown list provided by the system.

Pros:
- Perfect data consistency: Zero typos, making dashboard filtering flawlessly reliable out of the box.
- Easier backend handling: No need for string parsing or Regex extraction.

Cons:
- Very rigid: Frustrating for users if their specific problem doesn't fit neatly into any of the predefined buckets.
- High maintenance: Requires continuous administrative effort to keep the list of categories up to date with the actual curriculum.
